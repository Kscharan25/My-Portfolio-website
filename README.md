<h1>Personal Portfolio Website</h1>

<p>
This project is a fully responsive and modern personal portfolio website designed to highlight my professional background, technical skills, and completed projects. The website follows a clean, structured layout with smooth animations, mobile-first responsiveness, and an integrated contact form.
</p>

<h2>Tech Stack</h2>
<ul>
  <li><img src="./images/icons/html.png" width="20"> HTML5 – Structure and semantic layout</li>
  <li><img src="./images/icons/css.png" width="20"> CSS3 – Styling, responsiveness, animations</li>
  <li><img src="./images/icons/js.png" width="20"> JavaScript – Interactivity, DOM updates, form validation</li>
  <li><img src="./images/icons/git.png" width="20"> Git & GitHub – Version control and project hosting</li>
</ul>

<h2>Features</h2>
<ul>
  <li>Responsive UI with adaptive layouts for desktop, tablet, and mobile</li>
  <li>Glassmorphism-inspired design with smooth reveal animations</li>
  <li>Sticky navigation bar with mobile sidebar menu</li>
  <li>Interactive hero section with resume download option</li>
  <li>Showcase sections for About, Tech Stack, Projects, and Contact</li>
  <li>Validating contact form integrated using FormSubmit</li>
  <li>Scroll progress indicator and section-based animations</li>
</ul>

<h2>Project Structure</h2>
<ul>
  <li><strong>index.html</strong> – Main page structure</li>
  <li><strong>style.css</strong> – Layout, animations, media queries</li>
  <li><strong>index.js</strong> – Navigation, resume actions, contact form logic</li>
  <li><strong>/images</strong> – Icons, illustrations, and assets</li>
</ul>

<h2>Setup Instructions</h2>
<ol>
  <li>Clone the repository</li>
  <li>Open <code>index.html</code> in any modern browser</li>
  <li>No additional configuration required</li>
</ol>
